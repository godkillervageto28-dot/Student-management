// What it does

Core Features

🔐
Role-Based Access

Three distinct roles — Admin, Faculty, Student — each with their own dashboard and permissions.

👤
User Management

Admins create Faculty and Student accounts on the fly. No backend or database required.

📊
Grade Tracking

Faculty update Math, Science, and English marks per student. Changes reflect instantly.

📅
Attendance Records

Track attendance percentage for every student with visual score bars.

💾
Persistent Storage

All data saved via localStorage — refresh the page without losing anything.

⚡
Zero Setup

Open the file and go. No server, no database, no install steps required.
