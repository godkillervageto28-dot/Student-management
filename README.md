Student Management System (SMS):

A simple 'frontend-based Student Management System' built using "HTML, CSS, and JavaScript". This project demonstrates role-based access for "Admin, Faculty, and Students" without using any backend or database.

Features:

Role-Based Login System:
  - Admin
  - Faculty
  - Student

Admin Panel:
  - Create new users (Faculty & Students)
  - View all users in the system

Faculty Panel:
  - View all students
  - Update attendance
  - Enter subject-wise marks (Math, Science, English)

Student Panel:
  - View personal profile
  - Check attendance percentage
  - View subject marks with visual indicators

Local Storage:
  - Data is stored in browser 'localStorage'
  - No backend required

Zero Setup:
  - Just open 'index.html' and start using


Technologies Used;

- HTML5
- CSS
- JavaScript
- Browser LocalStorage

How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on 'index.html' or open it in a browser
4. Click Login →
5. Use default admin credentials to start


 How It Works

1. Admin Login:
   - Login using default credentials

2. Create Users:
   - Add Faculty and Student accounts

3. Faculty Login:
   - Enter attendance and marks

4. Student Login:
   - View academic performance


Limitations:

- No backend (data stored only in browser)
- Data will be lost if:
  - Browser cache is cleared
  - Different browser/device is used
- No authentication security (basic demo only)
